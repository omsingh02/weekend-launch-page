# Weekend Launch Page

Personal one-page offer site for Om Singh's **Weekend Launch Page** service — fast, clean, mobile-first launch sites for student-led projects and small teams.

## What this is

A focused service page offering one-page launch site builds for:

- Student founders
- Hackathon teams
- College clubs & event organizers
- Tiny early-stage teams

## Sections

1. Hero with clear offer statement
2. Who this is for — audience self-identification
3. What I build — scoped service list
4. What's included / not included
5. Why this approach works — technical edge
6. Sample builds — demo project cards
7. About Om Singh
8. Final CTA
9. Footer with contact links

## Tech stack

- HTML
- CSS (custom properties, mobile-first)
- No frameworks, no build tools, no dependencies

## Run locally

```bash
# Any static file server works
python3 -m http.server 8080
# Open http://localhost:8080
```
